Zurb Foundation for Drupal 7
============================

This is a Drupal 7 theme built using Foundation by Zurb - http://foundation.zurb.com


Regions
==============

* Header
* Navigation
* Left
* Right
* Content
* Footer


Notes
=====

app.css and app.js need to be customized for your site.
